# TP1 - HTTP Client / Server (Java)

Projet réalisé dans le cadre du cours de Cloud Computing (UQAC).

## Contenu
- ex2-server : serveur HTTP minimal en Java (sockets TCP)
- ex1-client : client HTTP simple en Java

## Lancement du serveur
- Lancer la classe `HttpServerApp`
- Accéder via http://localhost:8080

## Test
- Le serveur renvoie index.html si le fichier existe
- Sinon, renvoie une page d’erreur (401)
